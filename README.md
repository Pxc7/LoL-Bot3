# Installation

##  _Termux_
```cmd
> pkg update && pkg upgrade
> pkg install git -y
> pkg install nodejs -y
> pkg install ffmpeg -y
```

## _Cloning_
```cmd
> git clone https://github.com/Pxc7/LoL-Bot3
> cd LoL-Bot3
```

## _Install_
```cmd
> npm i
```

## _Run the LoL-Bot3_
```cmd
> npm start
```
